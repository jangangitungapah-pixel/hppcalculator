# MVP Scope

## MVP Features
- **Quick HPP Calculator:** Single-page form to input ingredients, packaging, extra costs, yield, and selling price.
- **Result Dashboard:** Clear, readable output of HPP, profit, margin, and business health status.
- **Local Storage Saving:** Save calculation results locally in the browser.
- **History/Saved Calculations:** View previously saved calculations.
- **Bilingual Support:** Switch between Indonesian (default) and English UI.
- **Basic Settings:** Language toggle, simple currency/number format preferences.

## Non-MVP Features (Excluded from Phase 1)
- User Authentication (Login/Sign up)
- Cloud Database and Sync
- Complex Ingredient Database / Inventory
- Multi-step Recipe Costing
- AI Assistant
- PDF/Excel Export

## Future Features (SaaS Direction)
- Pro Tier with unlimited cloud saves
- Team collaboration and multiple locations
- Reseller and Marketplace pricing calculators (e.g., GoFood/GrabFood tax calculation)
- Advanced sales reporting and analytics

## Feature Priority Table

| Feature | Priority (MoSCoW) | Description |
| :--- | :--- | :--- |
| Quick Calculator Form | **Must Have** | The core input engine for costs and prices. |
| Result & Status Badge | **Must Have** | Display HPP, Profit, Margin, and "Safe/Loss" status. |
| Local Storage Save | **Must Have** | Ability to save without login. |
| Language Toggle (ID/EN) | **Must Have** | Accessible for both local and expat UMKM. |
| Dashboard Overview | **Should Have** | Quick shortcuts and recent calculation summaries. |
| Saved Items List | **Should Have** | View past calculations. |
| Basic Settings | **Could Have** | Change currency formats or default margin targets. |
| Ingredient Database | **Won't Have (MVP)** | Saved list of raw materials for reuse. |
| User Accounts | **Won't Have (MVP)** | Registration and login system. |

## Why Quick Calculator Comes First
To build trust with non-technical users, we must deliver immediate value with zero friction. Forcing users to build an ingredient database or create an account before they can calculate their first product will cause massive drop-offs. A "Quick Calculator" proves the app's value in under 60 seconds. Once they see the value, they will be more willing to adopt complex features in the future.
