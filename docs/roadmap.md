# Product Roadmap: From MVP to SaaS

## Phase 1: Product Planning (Current)
- **Goal:** Define the product vision, scope, users, and core logic.
- **Deliverables:** `/docs` planning files.
- **Acceptance Criteria:** All Phase 1 documentation is approved and finalized.

## Phase 2: Core Calculation Logic
- **Goal:** Implement the math and pure functions safely.
- **Features:** Typescript/Javascript utility functions for all HPP and margin calculations.
- **Deliverables:** Testable core logic modules, Unit tests.

## Phase 3: UX Wireframe & UI Design System
- **Goal:** Define the visual language.
- **Features:** Setup TailwindCSS/CSS styling, define color palettes, create base components (Buttons, Inputs, Cards).
- **Deliverables:** Reusable UI components library.

## Phase 4: Frontend MVP
- **Goal:** Build the React + Vite web application based on the MVP scope.
- **Features:** Welcome, Dashboard, Calculator, and Result pages. State management implementation.
- **Deliverables:** Fully functional React PWA (in-memory state).

## Phase 5: LocalStorage Persistence & i18n
- **Goal:** Save data and support languages.
- **Features:** LocalStorage hooks, multilingual dictionaries (ID/EN), Settings page.
- **Deliverables:** App remembers calculations and language across reloads.

## Phase 6: Advanced F&B Tools (Future)
- **Goal:** Introduce complex costing.
- **Features:** Ingredient database, Recipe builder, Marketplace fee calculator.
- **Deliverables:** Advanced calculator module.

## Phase 7: Backend & Auth Foundation (SaaS Prep)
- **Goal:** Move to the cloud.
- **Features:** Node.js/Go backend, PostgreSQL, Firebase/Supabase Auth, API development.
- **Deliverables:** Users can register and sync their local data to the cloud.

## Phase 8: SaaS Features & Monetization
- **Goal:** Generate revenue.
- **Features:** Pro tier, Stripe/Midtrans integration, advanced analytics, team sharing.
- **Deliverables:** Complete SaaS platform ready for paid subscribers.

## Phase 9: Reports & Exports
- **Goal:** Enhance business utility.
- **Features:** PDF generation, Excel export, monthly profit charts.
- **Deliverables:** Export features module.

## Phase 10: Production Release & Native Wrapping
- **Goal:** Expand reach.
- **Features:** Wrap PWA into Android (TWA/Capacitor) and iOS. App Store submissions.
- **Deliverables:** Live application on multiple app stores.
